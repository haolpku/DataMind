RAG All-in-One 项目说明

本项目是一个基于 LlamaIndex 框架构建的智能知识助手，采用 MVP 迭代方式开发。

== 当前功能 (MVP) ==

1. RAG 检索增强生成
   - 支持加载 PDF、TXT、MD、DOCX 等格式文档
   - 使用 Chroma 向量数据库存储文档向量
   - 支持语义检索，找到与用户问题最相关的文档片段
   - 基于检索到的内容生成回答

2. 对话记忆
   - 短期记忆: 保持当前对话的上下文连贯性
   - 长期记忆: 从历史对话中提取关键信息

3. 智能 Agent
   - 自动判断是否需要检索知识库
   - 可以进行自由对话，也可以精准查找文档信息

== 后续规划 ==

4. GraphRAG: 基于知识图谱的检索，处理多跳推理问题
5. Database: 自然语言查询数据库 (NL2SQL)
6. Skills: 自定义工具和技能扩展
7. Router: 多 Agent 协作编排

== 技术栈 ==

- 框架: LlamaIndex
- 向量库: Chroma (本地, 无需 Docker)
- LLM: OpenAI 兼容 API (DeepSeek/智谱/月之暗面等)
- 语言: Python 3.10+
