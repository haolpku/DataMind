# 2026 Q2 远程办公政策

**生效日期**：2026-05-01
**起草**：HR（Olivia） · **法务复核**：Queenie · **批准**：CEO
**适用范围**：Acme Technologies 全员（含外包人员的特殊条款见 §6）

## 1. 核心原则

Acme 默认是**线下办公优先**的公司——核心业务靠面对面协作驱动。但我们承认远程办公在以下场景能显著提升产出与员工幸福感：

- 深度专注工作（设计文档、复杂 debug、长 SQL）
- 家庭事务（接送小孩、装修验收、就医陪护）
- 跨时区协作（与海外客户 / 团队沟通）

## 2. 标准模式

### 2.1 Hybrid（默认）
- 每周线下 ≥ 3 天
- 周三全员强制无会议（详见员工手册深度工作时段条款）
- 远程时段需在 Slack 状态注明 "remote" + 大致时段

### 2.2 全远程（特殊审批）
- 司龄 ≥ 2 年
- 直属 manager + VP 双重批准
- 试行期 3 个月，到期评估

### 2.3 临时远程
- 单次 ≤ 5 个工作日
- Manager 当日批复即可
- 累计单月不超过 8 天

## 3. 跨地域

- 员工跨城市工作（如上海 → 北京）需提前 1 周报备
- 跨城超过 14 天按"出差"处理，由财务（Peter）报销住宿
- 跨国（含港澳台）需要 HR + 法务审批，注意税务合规

## 4. 设备与网络

- 公司提供：MacBook Pro、4K 显示器、人体工学椅（家庭办公满 3 个月可申请）
- 网络：每月 200 元家庭宽带补贴
- VPN：连接 Vault / 内部 Grafana / Jira 必须走公司 VPN

## 5. 安全要求

详见 `security-policy.md` 第 3 章。远程额外要求：

- 公共 WiFi 必须走公司 VPN
- 笔记本离开视线立即锁屏
- 屏幕共享前确认无敏感信息
- 客户数据严禁本地存储

## 6. 外包人员条款

外包人员**不适用**本政策的 hybrid 标准，远程需求由项目 PM 与外包公司协商，按合同条款执行。

## 7. 违规处理

- 第 1 次：HR 面谈
- 第 2 次：书面警告 + 暂停远程权限 1 个月
- 严重违规（数据泄露、安全事件）：直接 PIP 或解雇

## 8. 反馈渠道

- 政策建议：#policy-feedback Slack 频道
- 个案咨询：HR Olivia
- 法务相关：Queenie

## 附录：常见问答

**Q: 我可以临时去咖啡店工作吗？**
A: 可以，但需登录公司 VPN，且不要打开客户数据相关界面。

**Q: 跨时区会议怎么排？**
A: 主 calendar 用上海时间，跨时区参与者自行切换。AI Copilot 团队（林涛 owner）跟海外客户多，可申请固定会议窗口。

**Q: 周三 deep work 时段一定不能开会吗？**
A: 紧急生产事故除外（参考 incident-response-sop.md），其他一律拒绝。
